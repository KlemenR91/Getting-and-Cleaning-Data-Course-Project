## Step by step transformations
* Download data
* Read data
* Merge Y, X and Subject collums for test and train data
* Merge test and train rows
* Create table with collums that names contains "mean" or "std" + activityID and subjectID
* Sort created table
* Clean names with gsub
* Merge activity with activity_labels
* Create table with the average of each variable for each activity and each subject
* Merge activity with activity_labels
